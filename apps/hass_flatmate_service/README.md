# Hass Flatmate Service App

This folder contains the Home Assistant App/Add-on packaging for the backend service.

Source mirror used for image build:
- `service_src/`

Canonical editable source:
- `../../addon/hass_flatmate_service/`

App changelog:
- `CHANGELOG.md`

Before building a new app image after backend changes, run:

```bash
./scripts/sync_app_service_src.sh
```
